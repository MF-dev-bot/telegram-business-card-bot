# 🤖 Telegram Бот-Визитка

Telegram бот — цифровая визитка компании с информацией об услугах, 
ценах, портфолио и контактах.

![Python](https://img.shields.io/badge/Python-3.11+-blue?logo=python)
![Aiogram](https://img.shields.io/badge/Aiogram-3.x-green?logo=telegram)

---

## 📋 Возможности

- ℹ️ Информация о компании
- 📂 Каталог услуг
- 💰 Прайс-лист с ценами
- 📞 Контакты и обратная связь
- 🕐 График работы
- ✉️ Заявка прямо из бота
- 👨‍💼 Уведомление админу о новой заявке

---

## 🛠 Технологии

| Технология | Назначение |
|---|---|
| **Python 3.11+** | Основной язык |
| **Aiogram 3.x** | Telegram Bot API |

## 🚀 Установка

### 1. Клонировать

```bash
git clone https://github.com/MF-dev-bot/telegram-business-card-bot.git
cd telegram-business-card-bot

2. Установить зависимости
bash

pip install -r requirements.txt

3. Настроить

Откройте bot.py и замените:
python

BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"  # Токен от @BotFather
ADMIN_ID = 123456789               # Ваш Telegram ID

4. Запустить
bash

python bot.py

📁 Структура

├── bot.py              # Основной код бота
├── requirements.txt    # Зависимости
└── README.md           # Документация

📸 Скриншоты

    Бот предоставляет удобное меню с кнопками для навигации
    по разделам: услуги, цены, контакты, заявка.
